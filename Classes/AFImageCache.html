Inherits from NSCache

Overview

AFImageCache is a subclass of NSCache that stores and retrieves images from cache.

@discussion AFImageCache is used to cache images for successful AFImageRequestOperations with the proper cache policy.

cacheImage:forURL:cacheName:

Stores an image into cache, associated with a given URL and cache name.

- (void)cacheImage:(UIImage *)image forURL:(NSURL *)url cacheName:(NSString *)cacheName

Parameters

image

The image to be stored in cache.

url

The URL to be associated with the image.

cacheName

The cache name to be associated with the image in the cache. This allows for multiple versions of an image to be associated for a single URL, such as image thumbnails, for instance.

Discussion

Stores an image into cache, associated with a given URL and cache name.
